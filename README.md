# kani-lunch

Lunch map where you can eat Echizen crab.

## Demo
https://code4fukui.github.io/kani-lunch/

## Features
- Interactive map showing restaurants that serve Echizen crab
- Displays restaurant details like address, phone number, menu, and capacity

## Data / API
The data is sourced from the [Fukui Prefecture Tourism Data Analysis System "FTAS"](https://www.fuku-e.com/feature/detail_266.html).
