# 越前がにが食べられるランチマップ

越前がにを提供しているお店の位置情報を地図上に表示するWebアプリです。

## デモ
https://code4fukui.github.io/kani-lunch/

## 機能
- 越前がにが食べられるお店の位置を地図上に表示
- お店の詳細情報（店名、住所、電話番号、営業時間、メニュー、定員、バス駐車場の有無など）を確認

## 必要環境
特に必要ありません。Webブラウザさえあれば利用できます。

## 使い方
Webブラウザで上記デモページにアクセスすることで、越前がにが食べられるお店の地図を確認できます。

## データ
このプロジェクトで使用しているデータは、「福井県観光データ分析システム「FTAS」」から取得したものです。
