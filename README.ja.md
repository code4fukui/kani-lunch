# kani-lunch - 越前がにランチマップ

福井県で冬の味覚「越前がに」を食べられる飲食店を表示するインタラクティブなウェブマップです。

## デモ

**ライブデモ:** https://code4fukui.github.io/kani-lunch/

## 機能

- **インタラクティブなマップ:** クリック可能なピンで店舗の位置を表示します。
- **詳細情報:** 以下の店舗詳細情報を確認できます:
  - 店舗名称、住所、電話番号
  - 営業時間と休み（定休日）
  - かに関連メニュー名および値段
  - 定員とバス駐車場の有無

## 仕組み

バックエンドを持たない静的なウェブアプリケーションです。[`csv-map`](https://github.com/code4fukui/csv-map)ウェブコンポーネントを使用し、`kani-lunch-echizen.csv`ファイルから店舗データを直接解析して、インタラクティブなマップ上に描画します。

## 使い方

インストールは不要です。最新のウェブブラウザでデモページにアクセスするだけで利用できます。

ローカルで実行する場合は、このリポジトリをクローンし、ブラウザで`index.html`を開いてください。

## データソース

店舗データは[福井県観光データ分析システム「FTAS」](https://www.fuku-e.com/feature/detail_266.html)から提供されています。
